# Chord Overrides Este directorio contiene `chord-overrides.js`, un archivo generado desde el **Modo Edición** del cancionero. **No editar manualmente.** Se genera automáticamente desde el editor con el botón "Exportar overrides". ## Cómo funciona 1. Edita acordes en el cancionero (Modo Coro → abrir acordes → Editar) 2. Guarda los cambios 3. Cuando estés conforme, usa "Exportar overrides" 4. Sube el archivo `chord-overrides.js` a esta carpeta en GitHub Los overrides tienen prioridad sobre los acordes inline del HTML.
